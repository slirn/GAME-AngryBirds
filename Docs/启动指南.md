# 愤怒的小鸟游戏启动指南

## 项目概述
这是一个基于 Phaser 3 + Matter.js 物理引擎开发的愤怒的小鸟游戏。

## 环境要求
- Node.js (建议 v18.0.0 或更高版本)
- npm (Node.js 自带)

## 启动步骤

### 1. 安装依赖
在项目根目录执行以下命令：
```bash
npm install
```

### 2. 启动开发服务器
执行以下命令启动开发服务器：
```bash
cd C:\刘津语\编程\Games\AngryBirds
npm run dev
```

### 3. 访问游戏
- 开发服务器启动后会自动打开浏览器
- 游戏地址：http://localhost:3000

## 其他命令

### 构建生产版本
```bash
npm run build
```
构建后的文件会生成在 `dist` 目录中。

### 预览生产版本
```bash
npm run preview
```

## 项目结构
```
AngryBirds/
├── assets/           # 游戏资源文件
├── src/              # 源代码
│   ├── objects/      # 游戏对象
│   ├── scenes/       # 游戏场景
│   ├── utils/        # 工具函数
│   ├── config.js     # 游戏配置
│   └── main.js       # 游戏入口
├── DOCS/             # 文档目录
├── index.html        # 游戏HTML入口
└── package.json      # 项目配置
```

## 技术栈
- Phaser 3 (游戏框架)
- Matter.js (物理引擎)
- Vite (构建工具)
